## üìù STEP-BY-STEP INSTRUCTIONS

### Adding Skills (8-12 items):
1. Login to admin panel
2. Click "Skills" in the left sidebar
3. Click "Add Skill" button (top right)
4. Enter Name and Icon class
5. Click "Save and add another"
6. Repeat for all skills
7. Click "Save" on the last one

### Adding Projects (2-4 items):
1. Click "Projects" in the left sidebar
2. Click "Add Project" button
3. Fill in all fields:
   - Title
   - Description (detailed)
   - Tech Stack (comma-separated)
   - GitHub Link
   - Live Link (optional)
   - Upload Image (screenshot of project)
4. Click "Save and add another"
5. Repeat for all projects
6. Click "Save" on the last one

### Adding Certifications (2-5 items):
1. Click "Certifications" in the left sidebar
2. Click "Add Certification" button
3. Fill in all fields:
   - Title
   - Issuing Authority
   - Issue Date
   - Certificate Link
4. Click "Save and add another"
5. Repeat for all certifications
6. Click "Save" on the last one
